對四種模型訓練結果進行混淆矩陣後(以預測答案對照真實答案)
SVM--正確：387筆  錯誤：13筆  準確率(Accuracy)：96.75%、精準度(Precision)：96.8%、召回率(Recall)：97.01% 、F1分數(F1-score)：96.88%   (四項機率平均值)
KNN--正確：366筆  錯誤：34筆  準確率(Accuracy)：91.5% 、精準度(Precision)：91.6%、召回率(Recall)：91.78% 、F1分數(F1-score)：91.64%   (四項機率平均值)
DT --正確：511筆  錯誤：89筆  準確率(Accuracy)：85.17%、精準度(Precision)：84.96%、召回率(Recall)：85.24% 、F1分數(F1-score)：84.95%   (四項機率平均值)
RF --正確：520筆  錯誤：80筆  準確率(Accuracy)：86.67% 、精準度(Precision)：86.8%、召回率(Recall)：86.81% 、F1分數(F1-score)：86.70%  (四項機率平均值)
準確率(Accuracy)單純算出預測正確的機率，但當訓練資料的目標變數不平衡時則會有問題。
精確率（Precision）是計算陽性的樣本中有幾個是預測正確的，則分別代表price_range預測0,1,2,3四種預測正確的機率。
召回率（Recall）是事實為真的樣本中有幾個是預測正確的，則是代表price_range答案分別為0,1,2,3時，樣本中有幾個是預測正確的。
F1分數(F1-score)是精確率與召回率的調和平均數。
支持度(?)
以這些性能指標做計算後，SVM的準確率為最高。因資料無缺失，否則SVM對缺失資料敏感，且SVM對非線性問題沒有通用解決方案，必須謹慎選擇Kernelfunction來處理，最後我是用linear做出最好的結果。
